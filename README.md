### About the Project
> A Chrome extension enhancing video player functionalities: 
>   - Control video speed
>   - Adjust sound up to 200%
>   - OCR for copying text in the video
>   - Clickable video links
>   - picture-in-picture
>   - Video controls for YouTube shorts
